# Real-Time MERN Stack Application Projects

<h5> 4. Edit profile</h5>
<h5> 5. White and Dark Mode</h5>

## Technology: 
### Frontend: ReactJS + Vite, TailwindCSS, Redux
### Backend: NodeJS, ExpressJS, MongoDB
### Socket Server: Sockent.io
